# Download Notes

Source: https://gitee.com/kit-miao/dm-imu

`git clone` and Gitee archive downloads required authentication or returned HTTP errors in this environment, so this folder was populated through the public Gitee contents/raw API.

Included for local reference:
- README.md
- DM-IMU-L1 user manual PDF
- 2D drawing PDF
- CAN example source files needed for protocol interpretation

The folder intentionally has no `.git` directory. Some generated Keil/Cube build artifacts and one STEP raw download were not mirrored because Gitee returned 403 or the files are not needed for the deployment-side IMU bridge.
